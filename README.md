# Salas

## Resumen
Salas es una aplicación que permite administrar salas de reuniones.

### Installing
* Generar artefacto
```
mvn clean install
```
* Lanzar aplicación
```
java -jar SalasApplication-1.0.0.jar
```
* [Acceder aplicación] (http://localhost:8080/salas/)
